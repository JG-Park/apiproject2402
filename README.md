# apiproject2402
# 프로젝트 배경 🏚️
 - 프로젝트의 목적은 사용자가 원하는 조건을 입력하면 해당 지역, 법정동, 또는 건물용도에 따른 부동산 시세를 제공하여, 사용자가 집을 선택하는 데 도움을 주는 것입니다. 
이를 통해 사용자는 쉽게 전·월세 실거래 정보를 확인하고, 위치별 시세를 비교하여 집을 선택하는 시간을 단축할 수 있습니다.
